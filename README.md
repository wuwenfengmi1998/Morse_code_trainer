#硬件
STM32L03x 32PinIO 48MHz 
Buzzer 带静音电路
3.5 Audio接口 输入与输出 输出电容耦合
12864OLED
EEPROM
编码器开关 开机电路
TYPEC USB 充放电路 带串口芯片
电池 电量检测电路

#软件功能
基础UI
莫尔斯电码输入识别 手动键与自动键
蜂鸣器播放 音频口输出 可静音
可设置速率
电池电量检测
